CDM  v0.9
Console Draw Motor, C library for quickly developing Windows console applications with advanced graphics by wrapping WinAPI functions in easier to use containers. https://github.com/komo97/CDM

 CCDMContextA context in which CDM will run. It contains the handle for the console window, color information buffers, character buffers, the size of the window inside it's rect and an event structure to get key events
 CCDMEventA structure containing the events required to check input inside CDM
 CCDMPaletteA palette for a CDMSurface. It contains the four types of pixel that the surface is able to be used
 CCDMPixelDataThe data of a single pixel
 CCDMSurfaceA representation of an image that is able to be interpreted by CDM
 CCDMTextA representation of a text object able to be interpreted by CDM
 CCDMTilesA structure that contains raw data for the console to process, most structures used in CDM require this component
